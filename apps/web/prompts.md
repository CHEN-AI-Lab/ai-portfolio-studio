## 一、角色形象提示词（即梦文字生图）

> 生成后下载为角色参考图，用于后续图生视频。

### #1 正面全身（角色参考主图）

```
一位26岁的中国青年男性，中短发自然下垂，五官清秀，
眉毛浓密，眼神温和有神。身穿深蓝色工装外套，
内搭白色圆领T恤，黑色修身长裤，白色运动鞋。
双手自然垂放身侧，站姿放松。背景为纯浅灰色，
棚拍布光均匀。写实风格，电影级画质，8K细节，
面部光照柔和，无阴影遮挡。
```

### #2 45度侧面（角色参考辅助）

```
同一人物，深蓝色工装外套，白色T恤，黑色长裤，
45度侧身站立，微微转头看向镜头。
半身构图（腰部以上）。温色侧光从右前方打来，
面部立体感强，眼神平静专注。
背景纯浅灰色。写实摄影风格，电影级质感。
```

### #3 正面特写（表情参考备选）

```
同一人物，面部特写（胸部以上），微微仰头，
嘴角带一丝微笑，眼神望向远方若有所思。
深蓝色工装外套领口可见。柔和的暖色主光，
眼神光清晰。背景虚化浅灰色。电影级浅景深，
肤质真实自然，无明显磨皮感。
```

## 二、场景图提示词（即梦文字生图）

> 生成后作为背景参考，让图生视频时场景一致。

### 场景1：深夜工作室

```
一间深夜的独立工作室，窗户外是城市夜景，
窗玻璃上有雨滴痕迹。室内一张木制工作台，
桌上放着一台亮着屏幕的笔记本电脑、一盏暖色台灯、
一个陶瓷咖啡杯、几本书。背景墙上有几张
AIGC生成的画作打印品钉在软木板上。整体氛围
温暖、安静、专注。暗调电影感光线，只有台灯和
屏幕光作为主光源。构图：相机在房间另一侧，
透过工作台看到窗外的夜景。超写实，8K。
```

### 场景2：古城夜巷

```
西安古城墙脚下的一条夜间小巷，青石板路面有积水，
倒映着暖黄色的路灯和红色灯笼。两侧是古建筑风格的
店铺，门窗透出暖光。远处隐约可见古城楼的轮廓。
天空深蓝色，有薄云。巷子里空无一人，安静、
有故事感。氛围潮湿宁静。镜头位置：巷子中段，
稍微仰拍，前景石板路，中景两侧古建筑，
远景城墙。电影级构图，胶片质感，暖色补光。
```

### 场景3：城市天台夜景

```
一座高楼天台，前景是锈迹斑斑的金属围栏。
远景是西安城市的夜景灯光——远处有大雁塔的
轮廓和现代高楼的天际线。天空是深蓝色到紫色
的渐变，有稀疏的星星。天台上有一把旧木椅，
旁边放着一盏暖色露营灯。微风感觉。构图：
镜头低角度，前景围栏虚化，中景椅子和灯，
远景城市天际线。电影感，蓝紫调色盘，
平静、开阔的氛围。
```

## 三、图生视频提示词（即梦图生视频）

> 操作：上传角色参考图 + 场景背景图 → 勾选「角色参考」→ 粘贴下方 prompt

### 场景1：工作室创作（约10秒）

**上传素材**
- 角色参考图：正面全身 #1
- 场景参考图：深夜工作室

**Prompt**

```
[角色参考开启]
青年坐在工作台前，手指在键盘上敲击，
目光专注看着屏幕。台灯的光线温暖，
他微微侧身拿起咖啡杯喝了一口，
放下杯子后继续打字，嘴角带着满足的微笑。
窗外夜景和玻璃上的雨滴清晰可见。
镜头缓慢推进，从房间中部推到人物半身。
电影感运动，自然的细微动作（眨眼、轻度呼吸感），
24fps风格。时长10秒。
```

### 场景2：古城漫步（约12秒）

**上传素材**
- 角色参考图：侧面半身 #2
- 场景参考图：古城夜巷

**Prompt**

```
[角色参考开启]
青年走在城墙下的石板路上，步伐从容。
他穿着深蓝色工装外套，双手插在口袋里，
偶尔抬头看向城墙和灯笼。路过一处水洼时
低头看了一眼倒影，然后继续前行。暖色路灯
从上方洒下，影子在地面上拉长。镜头跟拍，
从背后稍侧的角度跟随人物走动。自然的行走
节奏，人物从画面右侧走向左侧。电影级浅景深，
前景有灯笼虚化。时长12秒。
```

### 场景3：天台眺望（约10-12秒）

**上传素材**
- 角色参考图：正面特写 #3
- 场景参考图：城市天台夜景

**Prompt**

```
[角色参考开启]
青年站在天台围栏前，双手轻轻搭在栏杆上，
望着远方的城市天际线和灯火。一阵风吹过，
他的头发和外套领口微微飘动。他深吸一口气，
脸上浮现出平静而坚定的表情。镜头从侧面
缓慢环绕约15度，展示人物和背景城市的关系。
远景中灯光闪烁。电影感调色，蓝紫夜空色调，
微风吹动的自然感。时长10-12秒。
```

## 四、剪映制作清单

### A. 背景音乐

| 场景 | BGM类型 | 剪映搜索关键词 |
|------|---------|---------------|
| 全程 | 氛围电子 | lo-fi 中国风 / 电子国风 氛围 |
| 场景1 | 轻钢琴 | 钢琴 深夜 安静 |
| 场景2 | 鼓点渐入 | 中国鼓 电子 / trap 古风 |
| 场景3 | 弦乐铺底 | 弦乐 氛围 希望 / cinematic 希望 |

> 推荐方案：找一首 3 分钟以内「国风电音氛围曲」全程使用，最省事。

| 场景 | 音效 | 剪映搜索关键词 | 音量建议 |

### C. 字幕/片头片尾

| 位置 | 文案 | 说明 |
|------|------|------|
| 片头 | **古城新梦** | 定格3秒，黑体加粗，字间距大，白色 |
| 片头副标题 | AIGC 短片创作 | 片头下方，细体灰色 |
| 场景1 | 无字幕 | 纯画面叙事 |
| 场景2 | 无字幕 | 纯画面叙事 |

| 片尾 | 制作工具：即梦 / ComfyUI | 定格3秒 |
| 片尾第二行 | Author：[你的名字] | 同上 |

### D. 转场

| 位置 | 类型 | 时长 |
|------|------|------|
| 场景1 → 场景2 | 交叉溶解 / 模糊 | 0.5秒 |
| 场景2 → 场景3 | 叠化 | 0.5秒 |

### E. 调色滤镜

| 场景 | 推荐滤镜 | 强度 |
|------|---------|------|
| 场景1 | 暖阳 / 胶片 | 50%-70% |
| 场景2 | 夜景 / 青橙 | 60%-80% |
| 场景3 | 蓝调 / 赛博 | 50%-70% |

> 剪映搜滤镜名，预览选顺眼的，统一视觉风格。

## 五、完整操作流程

```
Step 1: 文字生图 → 3 张角色卡（正面/侧面/特写）
Step 2: 文字生图 → 3 张场景图（工作室/古城/天台）
Step 3: 图生视频 → 场景1（角色#1 + 场景#1 + 勾选角色参考）
Step 4: 图生视频 → 场景2（角色#2 + 场景#2 + 勾选角色参考）
Step 5: 图生视频 → 场景3（角色#3 + 场景#3 + 勾选角色参考）
Step 6: 下载 3 段视频 → 传到电脑
Step 7: 剪映导入 → 加 BGM → 加音效 → 加字幕 → 加转场 → 调色
Step 8: 导出 → 上传到作品集网站（AI漫剧分类）
```
