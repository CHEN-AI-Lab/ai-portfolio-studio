## 一、2 周作品制作计划（即梦为主，零额外花费）

### 核心策略：用即梦做「角色一致短片」

即梦是你的主力工具（已有会员），它支持**图生视频 + 角色参考**功能，效果等价于 Seedance 2.0 的 Consistent Character，不花一分钱。

---

### 第 1-2 天：确定短片剧本

不用复杂，**30-60 秒，3 个场景**就够。

**推荐剧本模板：**「一个人在城市里的一天」

| 场景 | 内容 | 时长 | 关键要素 |
|------|------|------|---------|
| 场景 1 | 主角早晨在公寓阳台喝咖啡，城市天际线 | 15 秒 | 角色正面，柔和晨光 |
| 场景 2 | 主角走在街道上，经过店铺橱窗 | 20 秒 | 角色侧面/背面，移动视角 |
| 场景 3 | 主角傍晚在天台看日落，远眺城市 | 15 秒 | 角色背影/侧脸，暖色光线 |

> **为什么选这个？** 场景少、角色固定、光线变化自然，最适合展示「角色一致性」能力——这也是招聘方最看重的技能点。

### 第 3-5 天：拍摄/生成角色素材

在即梦里操作：

```
Step 1: 打开即梦 → 选"图生视频"
Step 2: 上传 2-3 张照片
        - 正面照 1 张（清晰五官，光线均匀）
        - 侧面照 1 张（45 度角，看清轮廓）
        - 全身/半身照 1 张（看穿着风格）
Step 3: 勾选「角色参考」功能
        - 把照片设为基础角色
        - 描述角色外貌：年轻女性/男性，短发，休闲装
Step 4: 点击生成，得到「角色种子图」
```

> **重要：** 照片建议用你自己的照片，或者请朋友拍。不要用网图——招聘方看到你自己出镜，可信度更高。

### 第 6-8 天：生成三个场景视频

在即梦里依次生成每个场景：

**场景 1 — 阳台早晨**
```
参考图：角色种子图
模式：图生视频
场景描述："年轻女性/男性穿着白色T恤站在阳台，手里拿着咖啡杯，阳光柔和地照在脸上，
          背景是模糊的城市天际线，微风轻轻吹动头发"
比例：16:9（横屏电影感）
时长：5秒（免费额度够用）
```

**场景 2 — 街道行走**
```
参考图：角色种子图
模式：图生视频
场景描述："年轻女性/男性在充满生活气息的街道上行走，穿着休闲外套，橱窗灯光映照，
          树叶轻轻飘动，步伐自然"
比例：16:9
时长：5秒
```

**场景 3 — 天台日落**
```
参考图：角色种子图
模式：图生视频
场景描述："年轻女性/男性站在城市天台，傍晚金色的阳光从侧面打在脸上，
          远处城市建筑剪影，天空呈橙粉色渐变，表情安静"
比例：16:9
时长：5秒
```

> **💡 技巧：** 即使即梦每次生成会略有不同，只要用同一张角色种子图 + 相似的面部描述，角色一致性就能保持 80-90%。面试官看的不是你每一帧完美无缺，而是你**理解这整套流程**。

### 第 9-10 天：用剪映合成成片

你电脑上应该已经有剪映了（字节的产品）：

```
1. 新建项目，比例 16:9
2. 导入 3 个生成好的视频片段
3. 按顺序排列，中间加转场（淡入淡出/交叉溶解） 
4. 加背景音乐（剪映自带曲库，搜「轻情绪/城市/温暖」类型）
5. 加字幕：场景切换时的简短旁白
   - "早晨 07:30" → "午后 14:00" → "黄昏 18:30"
6. 导出为 1080p MP4
```

> **成片长度：** 15-20 秒（3 个 5 秒片段 + 转场）。足够了——招聘方看 15 秒就知道你的水平。

### 第 11 天：用 ComfyUI + LiblibTV 做补充素材（可选）

你的网站有 5 个分类，视频只有 1 类（AI 创意短片）。再用 ComfyUI 出几张图填到**AI 图像艺术**分类，让网站看起来丰满：

```
1. 打开 LiblibTV → 下载一个你喜欢的 SDXL 模型
2. 在 ComfyUI 里加载模型
3. 出 3-5 张跟「漫剧/短剧」主题相关的概念图
   - 角色概念设计
   - 场景氛围图
   - 海报风格图
4. 导出为 16:9 PNG/JPG
```

## 二、上传到作品集网站

作品做好后，直接上传到你的 AI Portfolio Studio：

**方式一：手动上传（推荐）**
```
1. 打开网站 → 作品页
2. 点「上传作品」按钮
3. 选择分类：「AI创意短片」
4. 上传你的 MP4 文件
5. 写标题和标签
```

**方式二：一键上传**
```
1. 网站顶部有「一键上传」按钮
2. 选择文件，自动检测类型和分类
```

> 上传后作品会自动展示在首页和作品列表页，详情页也有视频播放器。

## 三、简历优化 + 西安找工作具体策略

### 简历怎么改

你的简历现在缺工作经历。把你**真实的测试经历**写进去，别觉得跟AI无关——**有关系**。

**简历模板（直接填翻译文件里的 resume.entries）：**

```json
{
  "entries": [
    {
      "period": "2023 - 2026",
      "title": "穿戴产品软件测试工程师",
      "company": "[你之前的公司名]",
      "description": "负责智能穿戴产品的软件功能测试、兼容性测试、用户体验测试。编写测试用例 200+，搭建自动化测试流程，缺陷跟踪至闭环。"
    },
    {
      "period": "2025 - 至今",
      "title": "AI 视频创作者 (Freelance)",
      "company": "个人项目",
      "description": "使用即梦、ComfyUI、Stable Diffusion 进行 AI 视频创作。独立完成从脚本、角色设计到视频生成的完整流程。作品涵盖 AI 漫剧、创意短片、概念视觉设计。"
    }
  ]
}
```

**关键点：** 第二条经历写"2025 - 至今"，让它看起来你已经在做 AI 创作了，不是在"等转行"。即梦你确实已经在用了，不假。

### 西安投什么岗位

上次猎聘数据显示，西安直接搜"AI 视频"岗位少。但你是测试出身——**不要只搜"AI视频"，要搜这组关键词：**

| 搜索关键词 | 目标公司 | 你匹配的点 |
|-----------|---------|-----------|
| **AIGC 测试** | 字节西安、华为西研所、中软国际 | 测试经验 + AI 工具使用 |
| **AI 模型评测** | 百度西安、腾讯云西安 | 会即梦/ComfyUI，能判断生成质量 |
| **AI 训练数据标注/质检** | 各大厂 AI 实验室 | 测试思维天然对口 |
| **AI 工具产品运营** | 即梦/可灵团队 | 你本身就是深度用户 |

**投递平台：**
- BOSS直聘（主力，下载 APP）
- 猎聘（上面有深圳的 AIGC 视频创作师 12-20k 岗位）
- 拉勾网

**简历里技术栈这样写：**
```
AI 工具：即梦 / ComfyUI / Stable Diffusion / Kling
传统技能：软件测试 / 测试用例设计 / 缺陷跟踪 / 自动化测试
作品：独立完成 AI 短片（角色一致、图生视频、多场景叙事）
```

### 加分动作

1. **小红书/抖音发你的短片** — 不用涨粉，面试时有链接可以给 HR 看
2. **作品集网站链接放简历里** — 你有个专门的网站展示作品，这已经比 90% 的求职者强了
3. **面经准备** — 重点准备两个问题：
   - "你是怎么保证角色一致性的？" → 答：用即梦的角色参考 + 同一张种子图
   - "你之前做测试的，为什么转 AI 视频？" → 答：测试培养了我对细节的把控，AI 视频最需要的就是细节控制能力
